#  React Weather App

A simple and modern weather app built with **React** and **Material UI (MUI)** that allows users to search for real-time weather information by city name using the **OpenWeatherMap API**.

---

##  Features

  - Search weather by city name
  - Displays:
  - Current Temperature
  - Min/Max Temperature
  - Humidity
  - Feels Like
  - Weather Description
  - Error handling for invalid cities
  - Stylish UI with Material UI and responsive design

---

##  Tech Stack

- React
- JavaScript (ES6+)
- Material UI (MUI)
- OpenWeatherMap API


---

##  Installation

```bash
# Clone the repo
git clone https://github.com/your-username/weather-app.git

# Go into the project directory
cd weather-app

# Install dependencies
npm install

# Run the app
npm run dev
```
